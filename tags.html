Tags

As tags são codigos que que envolvem um elemento dando-lhes atributos ou comportamento, as tags são a base de um documento html que por sua vez são a base para a forma que o usuario da internet vai receber esse conteudo.

Explicação das partes: DOCTYPE html→ Defina que o documento usa HTML5. html lang="pt-BR" → Indica que a página está em português do Brasil. head Contém metadados, como título da página e conjunto de caracteres. title Defina o nome da aba do navegador. body Contém o conteúdo visível da página. São tags para título e parágrafo : h1, p São tags para título e parágrafo.
