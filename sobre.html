Sobre o Html

O html é um acronomo para hyper text markup language, ou seja linguagem de marcação de texto, em portugues. Ele serve para demarcar e cofugurar uma pagina em um sitio eletronico (site), ele é frequente mente confundido com liguagem de programação, porem não é, e sim, de marcação como o acronomo dele descreve. Para demarcar um site usamos as tags(marcadores), para dizer como os elementos devem se comportar ou estruturar e organizar essa pagina.

O html é formado basicamente por três partes estruturalmente falando. São elas:

  • HTML
  • HEAD
  • BODY
